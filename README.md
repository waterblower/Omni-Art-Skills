# 通用创作技能索引

## 技能列表

1. [`creative-production-pipeline`](creative-production-pipeline/SKILL.md)  
   从故事/设定到剧本、分镜、资产清单、图片计划和关键帧的通用导演制片流水线。

2. [`script-to-shot-table`](script-to-shot-table/SKILL.md)  
   把剧本、故事梗概、口播稿、广告文案或小说段落转成面向 AI 图片、视频和漫画生成的 Markdown 镜头组；单一分镜只作为组内辅助节拍。

3. [`image-art-direction`](image-art-direction/SKILL.md)  
   图片美术风格导演：生成/选择参考图前的风格分析、候选图审查和生成后质检。

4. [`character-reference-pipeline`](character-reference-pipeline/SKILL.md)  
   角色参考图入库：按角色重要性制作头部正面、头部背面、立绘正面、立绘背面等标准资产。

5. [`image-generation-plan`](image-generation-plan/SKILL.md)  
   把故事、剧本、镜头组或制作包整理成逐张图片生成计划，不提交生成任务。

6. [`video-prompt-director`](video-prompt-director/SKILL.md)  
   视频生成提示词导演：把剧情/分镜转成简洁、可视、可听、可执行的视频提示词。

7. [`manga-creation-pipeline`](manga-creation-pipeline/SKILL.md)  
   漫画创作与漫画页生成流程：从方向、故事到镜头组、分格页和漫画图提示词。

8. [`reverse-image-prompt`](reverse-image-prompt/SKILL.md)  
   根据参考图反推高精度图片生成提示词，只输出提示词，不生成图片。
