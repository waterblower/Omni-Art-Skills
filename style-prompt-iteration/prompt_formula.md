# Prompt Formula

```text
[BASE_STYLE]
[SCENE]
[LIGHT]
[MOOD]
[SUBJECT]
[COMPOSITION]
[QUALITY]
[NEGATIVE_PATCH]
```

## Example

```text
[BASE_STYLE]
modern Japanese editorial anime illustration, painterly digital painting, matte color blocks, loose hand-painted texture, thin sketchy contour lines, soft blended shading, muted cinematic palette, subtle grain

[SCENE]
plain neutral background, simple readable space

[LIGHT]
soft diffused light, low-to-medium contrast

[MOOD]
muted blue-gray and warm coral accent palette

[SUBJECT]
simple test subject for this iteration

[COMPOSITION]
clear readable framing

[QUALITY]
key visual quality, clean finish

[NEGATIVE_PATCH]
not photorealistic, not 3D render, not glossy plastic, not hard vector lineart, not over-detailed
```
