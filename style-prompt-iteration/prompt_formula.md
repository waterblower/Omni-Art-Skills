# Prompt Formula

```text
[BASE_STYLE]
[CONTENT]
[LIGHT_COLOR]
[COMPOSITION]
[NEGATIVE]
```

## Example

```text
[BASE_STYLE]
modern Japanese editorial anime illustration, painterly digital painting, matte color blocks, loose hand-painted texture, thin sketchy contour lines, soft blended shading, muted cinematic palette, subtle grain

[CONTENT]
simple test subject for this iteration, plain neutral background, simple readable space

[LIGHT_COLOR]
soft diffused light, low-to-medium contrast, muted blue-gray palette with warm coral accents

[COMPOSITION]
clear readable framing

[NEGATIVE]
photorealistic skin, glossy plastic hair, wet specular hair, hard vector lineart
```
