# 正反同框方图立绘生成子技能

## 定位

本文件是 `image-generation-plan` 的子技能分支，用于写或修订人物“正反同框全身立绘 / 正背立绘 / 1:1 方图立绘”任务。凡图片计划中出现以下任务，都先按本文件执行：

- `CH-*STAND`
- `*_立绘_.png`
- `*_立绘_1x1.png`
- `*_正背立绘_1x1.png`
- prompt 中出现“正反同框”“正面与背面左右并列”“正背立绘”“1:1方图”的人物资产任务

本子技能只负责资产参考立绘，不负责源头大头照、年龄状态脸、剧情概念剧照或视频提示词。

## 核心定义

正反同框方图立绘是人物资产参考图，不是氛围图、剧照、海报或定妆照。它的目的只有三个：

1. 锁定全身比例、体态和服装层级。
2. 同时给出正面和背面，便于后续视频 / 概念剧照保持服装连续性。
3. 用纯白背景隔离人物，不让场景、光效、手持物、背包或行囊污染资产。

## 必须先有的参考

生成正反同框立绘前，应先有：

1. **正式脸 / 源头脸 / 年龄状态脸**
   - 用于同一张脸。
   - 不能用概念剧照锁脸。

2. **服装图或明确服装设定**
   - 若角色服装复杂、跨集换装、身份阶段变化明显，应先生成服装图，再生成立绘。
   - 若没有单独服装图，prompt 必须把服装层次写清楚。

3. **年龄阶段明确**
   - 所有立绘都必须写明确视觉年龄或年龄段。
   - 如果立绘对应某个状态脸，年龄必须与状态脸一致。

## 任务格式

推荐 TOML：

```toml
[[tasks]]
id = "CH-角色-STAND"
target_image_path = "资产/人物/角色/角色_立绘_1x1.png" # todo
reference_image_paths = ["资产/人物/角色/角色_源头大头照_3x4.png"]
aspect_ratio = "1:1"
prompt = """【基础美术风格】ARRI ALEXA 35，Open Gate 4.6K，Cooke Anamorphic/i 40mm，T2.8，2.39:1
【具体内容】角色正反同框全身立绘，视觉年龄……，正面与背面左右并列，1:1方图，纯白背景，全身完整不裁切，两个人物比例一致，……
【生成要求】……；图1用于人物身份与同一张脸"""
```

硬规则：

- `aspect_ratio = "1:1"`。
- prompt 必须写“正面与背面左右并列”。
- prompt 必须写“1:1方图”。
- prompt 必须写“纯白背景”。
- prompt 必须写“全身完整不裁切”。
- prompt 必须写“两个人物比例一致”。
- prompt 必须写“双手空置自然下垂”或“手部自然放松贴近身体”。
- prompt 必须写“背部只呈现衣料、袈裟披挂和衣褶结构”。
- 本轮只要改了 prompt、reference、年龄、服装或构图约束，`target_image_path` 必须标为 `# todo`。

## Prompt 必写内容

每个正反同框立绘任务必须写清：

- 角色名和图类型：正反同框全身立绘 / 正背立绘。
- 明确视觉年龄或年龄段。
- 身份阶段：青年僧人、印度求法期成年僧人、归唐译经僧、唐代士族儿童、王族人物等。
- 正面与背面左右并列。
- 1:1方图。
- 纯白背景。
- 全身完整不裁切。
- 两个人物比例一致。
- 服装层级、鞋履、腰带、袈裟、披帛、甲胄等结构可读。
- 双手空置自然下垂或手部自然放松贴近身体。
- 背面只呈现衣料层次、袈裟披挂方式和背面衣褶。
- 本图用途：后续概念剧照 / 视频生成的人物服装参考资产。
- 参考图用途：图1用于人物身份与同一张脸；图2用于服装层级 / 材质 / 配饰。

推荐句式：

```text
【具体内容】<角色>正反同框全身立绘，视觉年龄<年龄>，<身份阶段>，正面与背面左右并列，1:1方图，纯白背景，全身完整不裁切，两个人物比例一致，双手空置自然下垂，<服装、鞋履、袈裟/披帛/腰带>结构清晰，背部只呈现衣料层次、袈裟披挂和背面衣褶，作为后续概念剧照与视频生成的人物服装参考资产。
【生成要求】画面呈现提示词明确写出的主体、服装、比例和正背关系；构图清晰，历史材质细节可读，人物身份、数量和画幅边界稳定；图1用于人物身份与同一张脸。
```

## 不写内容

正反同框立绘不要写成剧照或海报。人物正反同框立绘中，手部和背部只服务服装参考；不得出现手持道具、经卷、法器、念珠串、包袱、背包、行囊、背筒、背带或挂在背后的长条包裹。

不要写：

- 灰色影棚、纹理墙、暗背景、室内场景、户外场景。
- 电影光效、强体积光、夕阳逆光、戏剧化烟尘。
- 手持道具、经卷、法器、念珠串、包袱、背包、行囊、背筒、背带或挂在背后的长条包裹。
- 行走、战斗、回望、跪拜、讲法等剧情动作。
- 多个年龄阶段混在同一张立绘里。
- 画面中出现第三个人或群演。
- “不要裁切 / 避免裁切”等负向句式；用“全身完整不裁切”这种正向约束。
- negative prompt 或 `【避免】` 段。


## 参考图规则

- 立绘可以引用源头脸 / 年龄状态脸锁身份。
- 立绘可以引用服装图锁服装层级。
- 立绘不得引用剧情概念剧照来锁脸。
- 如果参考图中有场景或光效，prompt 不重复场景，只说明取人物身份、服装层级或材质。
- 图注必须写在 `【生成要求】` 中：
  - `图1用于人物身份与同一张脸`
  - `图2用于服装层级、材质和配饰结构`

## 多年龄段人物

同一人物多个年龄段的立绘应跟随脸部状态链路：

1. 青年源头脸 → 青年立绘。
2. 三十多岁状态脸 → 三十多岁立绘。
3. 中年状态脸 → 中年立绘。
4. 晚年状态脸 → 晚年立绘。
5. 儿童阶段最后做，且按儿童身高、头身比、服装和发式处理。

不要用中年立绘反推青年立绘。不要让不同年龄阶段共用同一张立绘，除非服装与体态确实无明显变化且用户明确要求复用。

## 项目例：玄奘

在《大唐西域记》项目中：

- 青年玄奘立绘引用青年源头脸，视觉年龄约二十七岁，刚离开长安、才出发西行。
- 印度时期玄奘立绘引用印度三十多岁状态脸和青年源头脸，视觉年龄三十多岁，服务健陀罗至曲女城求法阶段。
- 归唐中年玄奘立绘引用归唐中年状态脸和青年源头脸，视觉年龄约四十四岁，服务长安译场阶段。
- 晚年玄奘若需要立绘，引用晚年状态脸和青年源头脸，视觉年龄六十岁以上，服务玉华宫译场阶段。
- 回忆中的年轻玄奘直接使用青年源头脸与青年立绘，不另做“年轻玄奘幻影”立绘。
