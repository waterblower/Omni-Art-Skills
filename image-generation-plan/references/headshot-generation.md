# 大头照 / 源头脸生成子技能

## 定位

本文件是 `image-generation-plan` 的子技能分支，用于写或修订人物“大头照 / 源头脸 / 年龄状态脸”任务。凡图片计划中出现以下任务，都先按本文件执行：

- `CH-*FACE`
- `*_源头大头照_3x4.png`
- `*_源头脸部特写_3x4.png`
- `*_状态脸_3x4.png`
- 任何用于锁定同一人物脸部身份的近距离头像任务

本子技能只负责人物脸部身份锚点，不负责服装图、正反同框立绘、概念剧照或视频提示词。

## 生成顺序

1. **先源头脸，后状态脸**
   - 源头脸定义“这个人是谁”。
   - 年龄状态脸定义“同一张脸在另一个年龄阶段是什么样”。
   - 服装图、立绘、概念剧照不得反过来作为脸部身份来源。

2. **多年龄段人物先做年轻非儿童阶段**
   - 同一人物有青年 / 中年 / 老年阶段时，先做青年或更年轻的非儿童阶段作为源头脸。
   - 后续中年、老年只作为由源头脸派生的状态脸。
   - 不要用中年、老年版本反推青年版本。
   - 若同一人物同时有儿童、青年、中年 / 老年阶段，小孩最后做；儿童脸不得覆盖青年源头脸。

3. **儿童阶段单独处理**
   - 儿童图必须写明确视觉年龄，例如“视觉年龄8岁左右的男童”。
   - 若儿童与成年属于同一人物，儿童图可以参考成年源头脸保持五官连续性，但服装、发式、身份按儿童时期写。

## 路径与任务格式

脸部任务推荐：

```toml
[[tasks]]
id = "CH-角色-FACE"
target_image_path = "资产/人物/角色/角色_源头大头照_3x4.png" # todo
reference_image_paths = []
aspect_ratio = "3:4"
prompt = """【基础美术风格】ARRI ALEXA 35，Open Gate 4.6K，Cooke Anamorphic/i 40mm，T2.8，2.39:1
【具体内容】角色源头脸部特写，视觉年龄……，……
【生成要求】……"""
```

状态脸推荐：

```toml
[[tasks]]
id = "CH-角色-某年龄状态-FACE"
target_image_path = "资产/人物/角色/角色_某年龄状态脸_3x4.png" # todo
reference_image_paths = ["资产/人物/角色/角色_源头大头照_3x4.png"]
aspect_ratio = "3:4"
prompt = """【基础美术风格】ARRI ALEXA 35，Open Gate 4.6K，Cooke Anamorphic/i 40mm，T2.8，2.39:1
【具体内容】某年龄阶段角色状态脸部特写，视觉年龄……，由图1源头脸自然变化而来，……
【生成要求】……；图1用于源头脸与五官连续性，本图不得反向成为源头参考"""
```

规则：

- `aspect_ratio` 固定优先用 `3:4`。
- `target_image_path` 行尾状态：本轮只要改了 prompt、reference、年龄或身份约束，一律 `# todo`。
- 源头脸通常 `reference_image_paths = []`，除非用户明确要求从同一人物其他已批准阶段保持连续性。
- 状态脸必须引用源头脸；可视需要再引用上一个状态脸辅助连续性，但生成要求里必须说明源头脸才是身份根。

## Prompt 必写内容

每个脸部任务必须写清：

- 角色名和图类型：源头脸部特写 / 状态脸部特写。
- 明确视觉年龄或年龄段：例如“视觉年龄约二十七岁的青年僧人”“视觉年龄三十多岁的成年僧人”“视觉年龄六十岁以上的老年法师”。
- 正面近距离身份头像。
- 画面从头顶到锁骨下方少量肩部。
- 柔和虚化的低细节历史环境背景。
- 人物脸部清晰突出。
- 保留少量符合身份的衣领 / 上衣质感。
- 五官连续性：脸型、额头、眉骨、眉眼、鼻梁、唇线、下颌、气质。
- 本图的用途：源头身份锚点 / 某年龄状态参考。

推荐句式：

```text
【具体内容】<角色>源头脸部特写，视觉年龄<年龄>，<身份阶段>，正面近距离身份头像，画面从头顶到锁骨下方少量肩部，柔和虚化的低细节历史环境背景，人物脸部清晰突出；<脸型>，<额头/眉骨>，<眉眼>，<鼻梁/嘴唇/下颌>，神情<气质>，保留少量肩部和符合身份的衣领/上衣质感，服饰信息简洁，只辅助判断时代与身份，作为后续同一张脸的身份锚点。
【生成要求】单人正面近距离身份头像，身份锚点重点定义脸、头型、发式、五官、脖颈和少量肩部轮廓；画面裁切到锁骨下方少量肩部，背景柔和虚化，面部细节清晰，人物长相符合剧本身份与年龄。
```

## 不写内容

脸部源头任务不要承担完整服装、道具、场景或剧情信息。

不要写：

- 大面积肩胸服装。
- 复杂服饰层级。
- 显眼道具、法器、佛珠、经卷、武器。
- 清晰剧情环境、具体宫殿 / 讲堂 / 战场陈设。
- 多人同框。
- “像某角色 / 不像某角色”这类负向比较。
- negative prompt 或 `【避免】` 段。

如果发现上述内容，任务应重写并标为 `# todo`。

## 参考图说明

带参考图的脸部任务必须在 `【生成要求】` 写明：

- `图1用于源头脸与五官连续性`
- `图2用于上一年龄阶段的成熟度连续性`
- `本图不得反向成为源头参考`

如果参考图已经包含服装、场景或道具，prompt 不重复这些细节，只说明取脸部连续性。

## 项目例：玄奘多年龄段

在《大唐西域记》项目中，玄奘脸部链路固定为：

1. 青年源头脸：刚离开长安、才出发西行，视觉年龄约二十七岁。
2. 印度求法期状态脸：视觉年龄三十多岁，由青年源头脸自然成熟而来。
3. 归唐译场状态脸：视觉年龄约四十四岁，由青年源头脸和印度三十多岁状态自然成熟而来。
4. 晚年状态脸：视觉年龄六十岁以上，由青年源头脸和归唐中年状态自然老化而来。
5. 回忆中的年轻玄奘：直接引用青年源头脸，不另做“年轻玄奘幻影”角色资产。

## 检查清单

- [ ] 所有脸部任务都有明确视觉年龄或年龄段。
- [ ] 源头脸是该人物最合适的年轻非儿童阶段。
- [ ] 中年 / 老年 / 特殊时期状态脸引用源头脸。
- [ ] 儿童阶段没有覆盖青年源头脸；如需连续性，只参考成年源头脸。
- [ ] prompt 写了“头顶到锁骨下方少量肩部”。
- [ ] prompt 写了“柔和虚化的低细节历史环境背景”。
- [ ] prompt 只承担脸部身份，不承担完整服装、道具、场景或剧情。
- [ ] 带参考图的任务用图1 / 图2说明用途。
- [ ] 本轮修改过 prompt、reference、年龄或身份约束的任务标为 `# todo`。
