# 角色外貌差异锚点子技能

## 定位

本文件是 `image-generation-plan` 的子技能分支，用于批量人物资产、配角人物、群像角色或任何已经出现“同脸感”的人物生成计划。凡写或修订多个 `CH-*FACE` / `CH-*STAND` 任务，必须先建立角色外貌差异锚点，再写 prompt。

## 触发条件

出现以下任一情况时启用本子技能：

- 同一文件内有 3 个以上人物脸部任务。
- 用户指出“同脸”“长得太像”“都像同一个演员”。
- 多个 prompt 都使用“清瘦脸型、眉眼温和、沉静、安定、历史剧男性脸”等近似模板。
- 配角、官吏、僧人、士人、论师等身份相近，模型容易收敛到同一套默认脸。

## 先做外貌差异表

写人物 prompt 前，先在心里或文档中为每个角色分配独立锚点。每个角色至少 6 项不同，覆盖：

- 脸型：宽脸、瘦长脸、方脸、圆脸、颧骨高、下颌宽、下巴尖、面颊凹陷。
- 额头与发际：宽额、窄额、发际后退、鬓角花白、剃发轮廓、发髻形态。
- 眉眼：粗眉、稀眉、长眉、窄眼、眼距偏宽、眼袋、眼窝深、上眼睑厚。
- 鼻口：鹰钩鼻、宽鼻翼、鼻梁塌、鼻梁秀直、薄唇、厚唇、嘴角下垂、法令纹。
- 肤色与纹理：白净、黝黑、风沙晒痕、蜡黄、红润、皱纹、痘坑、斑点。
- 胡须与年龄痕迹：短须、长须、髭须、络腮、鬓角花白、眼角纹、额纹。
- 体态与气质：父辈稳重、官员审视、边地警觉、山寺朴拙、兄长照拂、书写者文雅。
- 地域 / 阶层特征：洛阳士族、边地官吏、蜀中山寺、高昌王庭、那烂陀学僧等。

## 反同脸写法

Prompt 中不要只写身份和气质，要写可见的几何差异。推荐结构：

```text
【具体内容】<角色>源头脸部特写，视觉年龄<年龄>，<身份>，正面近距离身份头像，画面从头顶到锁骨下方少量肩部，柔和虚化的低细节历史环境背景，人物脸部清晰突出；<脸型>，<额头/发际/鬓角>，<眉眼与眼距>，<鼻型>，<嘴唇/法令纹/下颌>，<肤色与纹理>，<胡须/皱纹/年龄痕迹>，<角色独有气质>，保留少量肩部和符合身份的衣领/上衣质感，作为后续同一张脸的身份锚点。
```

## 大唐西域记配角示例

- 父亲陈慧：宽脸，颧骨温厚，法令纹明显，眼袋，鬓角花白，短士人胡须，父辈沉稳。
- 郑善果：瘦长窄脸，窄眼，鼻梁长直，长须，薄唇，官员冷静审视感。
- 官吏：晒黑，方下颌，粗眉，眼距偏宽，风沙皱纹，边地警觉。
- 兄长长捷 / 长捷：比玄奘更宽额，脸部更柔和，圆润下颌，眼神照拂，不共享玄奘清瘦锐利骨相。
- 蜀中法师：中年，颧骨明显，眼窝更深，肤色偏暖黄，山寺法师的朴拙沉静。
- 辩机：青年书写者，脸型较窄，鼻梁秀直，薄唇，眉眼敏锐，文雅但有年轻弟子感。

## 生成后相似度复检

生成人物图不是写完任务就结束。每生成一张人物图后，必须立刻和“上一张已生成的人物图”做相似度复检；如果是批量生成，还要和本批已经通过的人物图逐张快速对照。

复检规则：

1. **不同角色不能长得像**
   - 只要不是同一个角色，脸型、眉眼、鼻型、嘴唇、下颌、肤色、胡须、年龄痕迹和整体气质都必须有足够辨识度。
   - 父子、母女、师徒、同门、同族也必须有清楚差异；可以有血缘或地域相似感，但不能像同一个演员换装。
   - 只有同一个角色的不同年纪、明确双胞胎/分身/幻影设定，才允许高相似度；即便如此也要标注原因。

2. **复检对象**
   - 新生成的 FACE 图：对比上一张 FACE 图，以及本批已通过的其他 FACE 图。
   - 新生成的 STAND 图：先看脸是否继承对应 FACE，再对比上一张 STAND / FACE 是否误用同一套脸。
   - 如果当前图是某角色年龄状态脸，只和该角色源头脸保持连续性，不和其他角色靠近。

3. **判定标准**
   - 若第一眼像同一个演员、同一张脸换发型/换服装，判为不合格。
   - 若脸宽、颧骨、眼距、眉形、鼻型、唇形、下颌、肤色和胡须中多数相同，判为不合格。
   - 若 prompt 已写差异但生成图没有体现，仍判为不合格。

4. **返工动作**
   - 把当前图片任务保留为 `# todo`。
   - 强化当前角色自己的正向外貌锚点：脸宽/脸长、颧骨、眼距、鼻翼、唇厚、胡须、皱纹、肤色、体态。
   - 不在 prompt 中写“不要像上一张”“避免同脸”；只写更明确的正向差异。
   - 返工后重新生成，并再次与上一张及本批已通过人物图对比。

5. **记录要求**
   - 如果生成执行器支持日志，在生成记录中写一句：`相似度复检：通过 / 返工，比较对象=<上一张或角色名>`。
   - 如果发现同脸，记录具体原因，例如“眼距、鼻型、下颌和唇形接近上一张郑善果”。

## 批量修订规则

- 同一批人物中，不得复用同一套“脸型清瘦、眉眼温和、气质沉静”的通用句。
- 每个 FACE prompt 必须至少有 4 个几何外貌锚点和 2 个年龄 / 纹理 / 胡须 / 肤色锚点。
- STAND prompt 必须继承该角色的脸部锚点，不只写“保持同一张脸”。
- 如果角色已经生成且同脸感明显，FACE 与对应 STAND 都标为 `# todo`。
- 角色差异写法只用正向描述，不在 prompt 中写“不要像某某”“避免同脸”。

## 检查清单

- [ ] 每个角色有独立外貌锚点。
- [ ] 每个 FACE prompt 写了脸型、眉眼、鼻型、嘴唇/下颌、肤色/纹理、年龄痕迹中的至少 6 项。
- [ ] 同一批角色没有连续复用同一组脸部形容词。
- [ ] 同脸返工角色的 FACE 和 STAND 都是 `# todo`。
- [ ] STAND prompt 继承角色自己的外貌锚点和气质。
- [ ] 每张新生成人物图已和上一张人物图 / 本批已通过人物图做相似度复检。
- [ ] 不同角色即使有血缘、师徒、同门关系，也有足够辨识度。
